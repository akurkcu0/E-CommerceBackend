  * SQL Server
  * N Layered Architecture (Entity, Data Access, Business, WebAPI, Core)

  * Entity Framework 
  * IoC
  * Autofac
  * FluentValidation (for building validation rules)
  * JWT (Authentication), Hashing and Salting
